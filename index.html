Project Highlights

My programming experience is primarily in Machine Learning applications. My personal projects tackle difficult problems in automatic music transcription, machine vision, NLP, and cancer diagnostics.

Deepsound

Deep Learning project that takes audio recordings like wave files and constructs the corresponding sheet music.

Icon

Smash-Hash

Nunc A two part neural network that first captions an image and then uses the caption embeddings to recommend similar images on a graph. This project is still in development and needs minor improvements to the captioning network and a reformulation of the Graphical Convolution Network.

Icon

BeyondMIDI

Deep Learning approach to music transcription with realistic sounding playback. This is a portion of my private Gitlab project named Deepsound @SimplySymplectic88. The project is still in development and I am still trying to figure out what I am willing to make public so stay tuned.

Icon

eoh-2018

Nunc This is an old project presented at UIUC for EOH-2018 that was last maintained by mlchrln2. Some demonstrations are included to get an idea of where this project was going. The rest of this project is private and has moved over to mlchrln2's GitLab account @SimplySymplectic88. Another spin off project is in the making in Github named BeyondMIDI. Check there for more updates on this project.

dnn-2018

Developed a deep neural network architecture that successfully classifies patients' tissue samples as cancerous or normal using labeled gene expression data generated by the Argonne National Lab (ANL).

physics_hackathon

HackIllinois 2018 project by UIUC physics students with various coding backgrounds. Our project was to create a Dungeons and Dragons (D&D) inspired AR mobile game with realtime geocoding.
