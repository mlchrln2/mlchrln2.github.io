Project Highlights

I strive to remain updated on modern developments in Machine Learning and Deep Learning. My personal projects tackle difficult problems in automatic music transcription, machine vision, NLP, and cancer diagnostics.

Icon

Deepsound

Automatic music transcription, music generation, and background noise reduction project.

Icon

Smash-Hash

Image recommendation network that captions images and searches images in a graph with similar styles and captions.

Icon

BeyondMIDI

Inspired by my Deepsound project, this spin off project applies some of key concepts from Deepsound to create music transcripts of clarinet recordings from audio files and create realistic sounding playback as a replacement for the MIDI format. This project is still not fully public so please be patient as I upload bits and pieces at a time.

Icon

eoh-2018

Originally presented as part of the physics department's Electronic Band exhibit at the Engineer Open House. This project is a visualize aid for individuals interested in the physics behind sound waves.

Icon

dnn-2018

Deep neural network architecture that successfully classifies patients' tissue samples as cancerous or normal using labeled gene expression data generated by the Argonne National Lab (ANL).

Icon

physics_hackathon

With the growing popularity of Live Action Role Playing (LARP) games, this project aimed to create a mobile version of Dungeons and Dragons with realtime geocoding. The final product after two days of working was presented at the HackIllinois 2018 event.

Icon

Neural-Networks

In order to fully understand every nuance that goes into neural network design and execution, this exploratory project attempts to reconstruct many of the utilities used by popular Deep Learning libraries such as Pytorch, Keras, and Tensorflow.
